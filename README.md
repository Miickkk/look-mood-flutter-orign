## Look & Mood 

Bem-vindo ao **Look & Mood**, seu aplicativo de guarda-roupa digital e assistente de estilo pessoal, desenvolvido com **Flutter**\!

Este projeto visa simplificar a organização do seu vestuário, ajudar você a montar *looks* inspiradores e registrar seu estilo diariamente, tudo isso com uma interface moderna e intuitiva.

-----

### Funcionalidades Principais

  * **Organização do Guarda-Roupa:** Catalogue suas peças de roupa com fotos, categorias, cores e estações.
  * **Montagem de Looks:** Combine diferentes peças para criar *looks* completos e salve-os para referência futura.
  * **Registro de Uso (Mood Tracker):** Registre quais *looks* você usou em determinados dias e o "mood" (humor/ocasião) daquele dia.
  * **Inspiração e Estatísticas:**
      * Veja quais peças são mais ou menos usadas.
      * Receba sugestões de combinações baseadas em seus itens e histórico.
  * **Interface Intuitiva:** Design limpo e focado na usabilidade, aproveitando os recursos do Flutter para uma experiência fluida.

-----

### Tecnologias Utilizadas

  * **Flutter:** Framework principal para o desenvolvimento nativo *cross-platform* (Android/iOS).
  * **Dart:** Linguagem de programação utilizada pelo Flutter.
  * **Gerenciamento de Estado (a definir, ex: Provider/Riverpod/Bloc):** Para uma arquitetura de código robusta e reativa.
  * **Banco de Dados Local (a definir, ex: Hive/Sqflite):** Para armazenar os dados do guarda-roupa e looks.
  * **Câmera/Galeria:** Utilização dos *plugins* nativos para captura e seleção de imagens das roupas.
  * **Firebase (Opcional):** Para autenticação, armazenamento em nuvem (Storage) e banco de dados *realtime* (Firestore/Realtime Database) - *Se aplicável*.

### Como Executar o Projeto Localmente

Siga estas instruções para configurar e executar o **Look & Mood** em seu ambiente de desenvolvimento.

#### Pré-requisitos

1.  **Flutter SDK** instalado e configurado (versão X.Y.Z ou superior).
2.  Um IDE de sua preferência (VS Code ou Android Studio) com os *plugins* Flutter e Dart instalados.
3.  Um emulador ou dispositivo físico conectado para teste.

#### Passos

1.  **Clone o repositório:**

    ```bash
    git clone https://docs.github.com/pt/migrations/importing-source-code/using-the-command-line-to-import-source-code/adding-locally-hosted-code-to-github
    cd look-and-mood
    ```

2.  **Instale as dependências:**

    ```bash
    flutter pub get
    ```

3.  **Execute a aplicação:**

    ```bash
    flutter run
    ```

    *(Certifique-se de que há um dispositivo/emulador conectado e pronto para rodar.)*
